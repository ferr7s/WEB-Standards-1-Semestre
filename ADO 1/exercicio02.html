Página Web com erros

O código HTML presente nesta página web tem sete erros. O desafio deste exercício é encontrar e corrigir esses erros e enviar a página correta junto com os demais exercícios desta ADO.

Repare se há alguma diferença entre a página errada e a corrigida. Se nada for diferente, significa que o Browser arrumou os erros. Mas será que ele conseguiu arrumar tudo?

Escrever o código de forma correta é uma responsabilidade de quem está criando a página, mas o Browser pode ajudar na maioria dos casos.
